# Frontend

Use the main setup guide at:

```text
../README.md
```

The frontend app lives in `frontend/`.

Create this local env file:

```text
frontend/.env.local
```

Required variable:

```env
API_URL=http://localhost:3001
```

Set `API_URL` to the backend URL from `backend/.env`.

Day-to-day commands:

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```
